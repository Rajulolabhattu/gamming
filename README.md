# Adaptive Snake (Android)

A lightweight Snake game for Android phones that scales the game board and HUD based on each device's screen size and density.

## Features
- Works from Android 5.0+ (`minSdk 21`).
- Automatically adjusts board cell size and dimensions to fit different resolutions.
- Swipe controls (up/down/left/right).
- Dynamic difficulty (snake speed increases with score).
- Tap to restart after game over.

## Build
```bash
./gradlew assembleDebug
```

## Run
Open this project in Android Studio and run it on a device or emulator.
